# 📚 GUI Кітапхана жүйесі (Library Management System)

## 🎯 Мақсаты:
Бұл жобаның мақсаты — GUI (графикалық интерфейс) мен CRUD (Create, Read, Update, Delete) операцияларын біріктіру. Пайдаланушы кітаптарды ыңғайлы түрде басқара алады.

---

## 📌 Жобаның сипаттамасы:
Бағдарлама графикалық интерфейс арқылы кітапхана жүйесін жүзеге асырады. Қолданушы:
- Кітап қоса алады  
- Кітапты өшіре алады  
- Кітапты іздей алады  

Жоба MVC (Model-View-Controller) архитектурасы негізінде құрылады.

---

## ❓ Сұрақтар және жауаптар:

### 1. MVC деген не?
MVC (Model-View-Controller) — бағдарламаны үш бөлікке бөлу архитектурасы:
- Model — деректер (кітаптар тізімі)  
- View — интерфейс (GUI)  
- Controller — логика (пайдаланушы әрекеттерін өңдеу)  

---

### 2. GUI мен модель қалай байланысады?
GUI (View) пайдаланушыдан мәлімет алады (мысалы, кітап атауы). Бұл ақпарат Controller арқылы Model-ге беріледі. Model өзгергеннен кейін, нәтиже қайтадан GUI-ға көрсетіледі.

---

### 3. Қандай компоненттер қолданылды?
GUI жасау үшін келесі компоненттер қолданылды:
- JFrame — негізгі терезе  
- JPanel — контейнер  
- JButton — батырмалар  
- JTextField — мәтін енгізу  
- JLabel — жазу көрсету  
- JList немесе JTable — кітаптар тізімі  

---

## 📊 Қорытынды:
Бұл жоба GUI мен логиканы бөлек басқаруды үйретеді. MVC архитектурасы арқылы код құрылымы түсінікті әрі ыңғайлы болады. Нәтижесінде толық функционалды кітапхана жүйесі жасалады.
